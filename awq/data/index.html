A3
ALTOIngerir extratos e validar pipeline end-to-endOperação

Impacto: Valida toda a cadeia de ingestão com dados reais.

A4
MEDIOEnforçar RBAC no middlewareSegurança

Impacto: cs-ops e analyst sem acesso a dados AWQ Group.

A5
BAIXOMigrar awq/page.tsx para consumir menos snapshotEngenharia

Impacto: Control tower mostrando dados reais em vez de snapshot.
